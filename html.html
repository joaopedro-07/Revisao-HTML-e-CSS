O que é HTML?

Definição básica

HTML é uma linguagem de marcação utilizada para criar e estruturar conteúdo na web, definindo elementos como texto, imagens, vídeos e links.

Estrutura

As páginas HTML são estruturadas com elementos como < html >, < head > e < body >, que organizam o conteúdo, metadados e elementos visíveis, respectivamente.

Tags e elementos

Utiliza tags (etiquetas) para marcar diferentes partes do conteúdo, como < p > para parágrafos, < img > para imagens e < a > para links.

Atributos

As tags podem ter atributos que fornecem informações adicionais sobre um elemento, como src para especificar a origem de uma imagem ou href para definir o destino de um link.

Semântica

HTML possui elementos semânticos que descrevem claramente o significado do conteúdo, como < header >, < nav >, < article > e (footer), melhorando a acessibilidade e a indexação por mecanismos de busca.

Padrões e evolução

HTML é mantido pelo W3C e sua evolução é documentada em especificações, como HTML5, que introduziu novos elementos e funcionalidades para melhorar a experiência na web.

A tag < ul > é usada para criar listas não ordenadas, enquanto a < ol > para criar as ordenadas. Itens dentro de uma lista são marcados com a tag < li >.
