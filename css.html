INÍCIO HTML CSS

O que é CSS?

Definição básica

CSS é uma linguagem de estilo utilizada para controlar a apresentação visual de páginas web escritas em HTML e XHTML.

Separação de conteúdo e estilo

CSS separa o conteúdo estrutural (definido em HTML) da apresentação visual (definida em CSS), permitindo maior flexibilidade e controle sobre o design das páginas.

Seletores e propriedades

CSS utiliza seletores para identificar quais elementos HTML serão estilizados, e propriedades para definir como esses elementos devem ser apresentados visualmente.

Cascata e herança

O termo "cascading" em CSS refere-se à hierarquia de regras de estilo, onde regras mais específicas têm precedência sobre regras mais gerais. Além disso, o princípio de herança permite que certas propriedades sejam aplicadas aos elementos filhos dos elementos estilizados.

Flexibilidade e responsividade

CSS oferece recursos poderosos para criar layouts flexíveis e responsivos, adaptáveis a diferentes dispositivos e tamanhos de tela, através de técnicas como media queries e unidades relativas.

Evolução e padrões

CSS é continuamente desenvolvido pelo W3C e pela comunidade web para incluir novos recursos e melhorar a compatibilidade entre navegadores. Versões mais recentes, como CSS3, introduziram novas funcionalidades como gradientes, animações e transformações.

Seletores de tag: são os seletores mais simples e básicos, permitem estilizar todos os elementos de uma determinada tag HTML.

p {
color: blue;
font-size: 16px;
}

Seletores de classe: permitem estilizar elementos que possuem um atributo class específico, são indicados com um ponto (.) seguido do nome da classe.

< p class="destaque" >Este é um parágrafo de destaque.< /p >

.destaque {
background-color: yellow;
padding: 10px;
}

Seletores de ID: são usados para estilizar um elemento específico que possui um atributo id único, são indicados com um hash (#) seguido do nome do ID.

< div id="cabecalho" > < h1 >Título do site< /h1 > < /div >

#cabecalho {
background-color: #333;
color: white;
padding: 20px;
}

Propiedades e valores

  • As propriedades são nomes que especificam quais aspectos dos elementos HTML serão estilizados.
  • Cada propriedade CSS define uma característica específica que pode ser ajustada, como cor, tamanho, margem, alinhamento, entre outros.
  • Exemplos de propriedades incluem color, font-size, margin, padding, background-color, border, text-align, entre muitas outras.
  • Os valores são atribuídos às propriedades para definir como essas características devem ser exibidas.
  • Cada propriedade tem um conjunto específico de valores que pode aceitar, como cores, tamanhos, unidades de medida, etc.
  • Exemplos de valores incluem blue (para cor), 16px (para tamanho de fonte em pixels), center (para alinhamento), #f0f0f0 (para código de cor hexadecimal),
    1em (para unidades de medida relativa ao tamanho da fonte), entre outros.
  • A sintaxe no CSS para aplicar propriedades e valores segue um padrão comum:

    selector {
    property: value;
    }
  • O selector especifica a qual elemento (ou elementos) HTML as regras de estilo devem ser aplicadas.
  • Dentro das chaves {}, são listadas uma ou mais declarações de estilo, onde cada declaração consiste em uma property seguida por : e seu value.

Classes e IDs

  • Classes são atributos HTML que podem ser aplicados a um ou mais elementos para identificá-los de forma específica.
  • Permitem reutilizar estilos em diferentes partes do documento HTML.
  • Em HTML, uma classe é especificada usando o atributo class dentro de uma tag.

    < p class="destaque" >Este é um parágrafo de destaque.< /p >

    Aqui, destaque é o nome da classe que será usada para estilizar o parágrafo.

  • Para estilizar elementos com uma classe específica, utiliza-se o seletor de classe (. seguido do nome da classe).

    .destaque {
    background-color: yellow;
    font-weight: bold;
    padding: 10px;
    }

  • IDs são atributos HTML usados para identificar de forma única um único elemento na página.
  • Cada ID deve ser único no documento HTML.
  • Em HTML, um ID é especificado usando o atributo id dentro de uma tag.

    < div id="cabecalho" >
    < h1>Título do site< /h1 >
    < /div >

    Aqui, cabecalho é o ID atribuído ao elemento < div >

    .
  • Para estilizar um elemento com um ID específico, utiliza-se o seletor de ID (# seguido do nome do ID).

    #cabecalho {
    background-color: #333;
    color: white;
    padding: 20px;
    }

Cascata e especificidade

  • A cascata refere-se ao processo pelo qual o navegador determina quais estilos devem ser aplicados a um elemento específico, levando em consideração várias fontes de estilos.
  • A especificidade refere-se à forma como o navegador determina qual regra CSS deve ser aplicada quando várias regras se aplicam ao mesmo elemento.
